# 🎮 Mini-Games Collection

This directory contains interactive mini-games that can be embedded in the GitHub profile.

## Available Games

### 🐍 Snake Game
- **File**: Automatically generated via GitHub Actions
- **Location**: `assets/images/github-contribution-grid-snake.svg`
- **Update Frequency**: Every 12 hours
- **Features**: Eats GitHub contributions, animated gameplay

### 🏓 Pong Game (Concept)
- **Implementation**: SVG-based animation
- **Features**: Auto-playing classic Pong game
- **Status**: Designed for future implementation with external service

### 🧩 Tetris Pieces (Concept)
- **Implementation**: Animated falling blocks
- **Features**: GitHub stats displayed as Tetris pieces
- **Status**: Planned integration with stats API

### 🎯 Interactive Quizzes
- **Implementation**: Click-to-reveal quiz questions
- **Features**: Tech knowledge testing with badges
- **Status**: Framework ready for content expansion

## Technical Implementation

All games are designed to work within GitHub's SVG rendering capabilities:
- Pure SVG animations
- No JavaScript dependencies
- Optimized for various screen sizes
- Accessible fallbacks included

## Customization

Games can be customized by modifying:
- Color schemes in SVG definitions
- Animation timing and duration
- Game difficulty parameters
- Integration with personal GitHub data
